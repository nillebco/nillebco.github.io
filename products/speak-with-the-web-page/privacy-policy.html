Privacy Policy

Last Updated: 2025-02-25

Thank you for using Speak with the Web Page ("the Extension"). This Privacy Policy explains how we handle your data when you use the Extension.

1. Data Collection and Usage

The Extension collects and processes the following data:

  • OpenAI API Key: The Extension requires your OpenAI API key to interact with the OpenAI real-time voice API. Your API key is stored in your browser’s local storage and is only used to send requests to OpenAI’s API. The Extension does not transmit or share this key with any other service or third party.
  • Voice Preference: The Extension stores your selected voice preference (among the ones available via OpenAI) in your browser’s local storage to improve your experience. This data is not shared externally.
  • Webpage Access: The Extension requires access to all visited webpages to enable its functionality, allowing you to interact with any page via voice. However, the Extension does not store or transmit your browsing history.
  • Microphone Access: The Extension requires microphone access to capture voice input. A consent prompt will be displayed, and without user approval, the Extension will not function. The recorded audio is sent only to OpenAI for processing and is not stored locally or transmitted elsewhere.

2. Data Storage

All stored data (API key and voice preference) is kept within the browser’s local storage. This data remains on your device and is not transmitted to any external servers beyond OpenAI’s API endpoints.

3. Data Sharing

The Extension does not collect, store, or share your personal data with any third parties. Your OpenAI API key and voice data are exclusively used for communication with OpenAI’s API.

4. Security

We take reasonable measures to protect your stored data within your browser. However, you are responsible for keeping your OpenAI API key secure, as it is stored locally on your device.

5. User Control

You can:

  • Delete or modify your OpenAI API key and voice preferences at any time through the Extension’s settings.
  • Revoke microphone permissions via your browser settings.
  • Disable or uninstall the Extension if you no longer wish to use it.

6. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected with a revised "Last Updated" date. We encourage users to review this policy periodically.

7. Contact Us

If you have any questions about this Privacy Policy, please contact us at hello@nilleb.com.
